# Reference Material

Reusable inputs for course-content generation and Copilot demos. Treat as authoritative — Copilot agents and skills are expected to consult these files before authoring.

| File | Purpose | When to use |
|------|---------|-------------|
| `microsoft-style-guide.md` | Microsoft Writing Style Guide (sentence case, bold UI labels, Oxford commas, input-neutral verbs, `should` vs `must`). | Authoring slide decks, exercises, code comments, exam-style questions, or any prose meant to match Microsoft house voice. |
| `fictional-companies.md` | Microsoft fictional companies (Contoso, Fabrikam, TailSpin, Trey Research, etc.). | Picking scenario stems for demos, exercises, or live coding so the room does not default to Contoso every time. |

Migrated from the GH-300 cert-prep sibling repo and relocated under `docs/references/` during the May 2026 layout reorg.
